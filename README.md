# Borys Online

Personal website and applications catalog built with Astro and hosted on GitHub Pages.

Factory integration notes:
- `FACTORY_INTEGRATION.md`

## Local Development

```bash
pnpm install
pnpm run dev
```

Visit `http://localhost:4321` to view the site locally.

## Build

```bash
pnpm run build
pnpm run check
```

## Deployment

Push to the master branch to deploy automatically via GitHub Actions.
