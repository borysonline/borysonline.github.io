## About ChromaMate

ChromaMate is a fast and intuitive color picker for iOS designers, photographers, and artists. Capture colors from your camera, edit them manually, and organize your favorite colors—completely free to use.

### Key Features

**Camera Color Picker**
- Real-time RGB and HEX color detection
- Tap anywhere to capture colors from your camera
- Magnifying lens for accurate picking
- Smart lighting detection

**Manual Color Editor**
- RGB, HSL, HSV, and HEX sliders
- Direct code input and editing
- Color wheel for intuitive selection
- CMYK and LAB color space support

**Smart Features**
- Copy to clipboard in multiple formats (HEX, RGB, HSL, HSV, CMYK, LAB)
- Automatic color name recognition (HTML Named Colors database)
- Universal search by color code or name
- Dark mode support
- Works completely offline - no internet required

**Core Features - All Free**
- Color palette creation and organization
- Unlimited saved colors
- Export colors in multiple formats
- Color naming and search
- Offline support

**Future Enhancements**
We may introduce optional premium features in the future, but all core functionality will remain free.
